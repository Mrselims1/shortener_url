------Turkish------
Bu Node JS Koduyla uzun linkleri kendi localinde oluşturduğun server ile kısaltabilirsin. 
Çalıştırmak için adımları takip et.
1. Node Js bilgisayarınızda kurulu olması gerekmektedir.
2. Verilen dosyayı çıkartıp terminal üzerinden "node index.js" yazmanız yeterlidir.
3. Terminal üzerinde sizin bilgisayarınız üzerinden sunucu açacak ve terminalde sizden uzun linki isteyecektir.
4. Uzun linki yapıştırdıktan sonra kısa linkinizi alabilirsiniz. 
not: Sadece sizin ağınızda olanlar için geçerlidir. 

-----ENGLISH-------
You can shorten long links using this Node.js code by creating a server on your local machine.
Follow the steps below to run it:

Node.js must be installed on your computer.

Extract the provided file and, in the terminal, simply type: node index.js

The terminal will start a server on your computer and prompt you to enter the long URL.

After pasting the long URL, you will receive your shortened link.

Note: This only works for users within your local network.
